## How to build
```
flutter clean
flutter pub get
flutter pub upgrade
flutter pub run change_app_package_name:main com.and96.rss_aggregator_flutter
flutter packages pub run flutter_launcher_icons:main
flutter build apk --release --no-tree-shake-icons
```
